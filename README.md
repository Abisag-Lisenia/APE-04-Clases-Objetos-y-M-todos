## 📌 Visión General

Este repositorio contiene la implementación de un sistema básico de registro estudiantil, diseñado para demostrar los principios fundamentales de la **Programación Orientada a Objetos (POO)**. El proyecto resuelve la automatización del cálculo de promedios y la determinación del estado académico de los estudiantes, garantizando la seguridad de los datos mediante un encapsulamiento estricto.

---

## 🚀 Características Principales

* **🔒 Encapsulamiento:** Protección de datos sensibles (cédula, notas) mediante modificadores de acceso `private`.
* **⚙️ Procesamiento Automático:** El constructor invoca métodos internos para calcular el promedio y el estado de aprobación en el momento exacto en que se instancia el objeto.
* **🔄 Mutabilidad Controlada:** Implementación de métodos `get` y `set`. Al actualizar una nota mediante un `set`, el sistema recalcula automáticamente el promedio y el estado para evitar inconsistencias de información.
* **🖥️ Despliegue Multi-Lenguaje:** Solución nativa desarrollada tanto en **C++**
